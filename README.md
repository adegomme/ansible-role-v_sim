# Ansible Role: marvel-nccr.v_sim

An Ansible role that installs [v_sim](https://v_sim.org) on Ubuntu.

## Installation

`ansible-galaxy install marvel-nccr.v_sim`

## Role Variables

See `defaults/main.yml`

## Example Playbook

  - hosts: servers
    roles:
    - role: marvel-nccr.v_sim
